Selectores Básicos

Cómo seleccionar elementos en el DOM para estilizarlos:


/* Selector de etiqueta */
p { color: blue; }

/* Selector de clase */
.mi-clase { background-color: red; }

/* Selector de ID */
#mi-id { font-size: 20px; }
