Cheatsheet HTML5

Guía rápida de referencia con la sintaxis y etiquetas esenciales de HTML.

Estructura Básica

Todo documento HTML5 debe incluir esta estructura mínima:


<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Título de tu página</title>
</head>
<body>
  <h1>Hola Mundo</h1>
</body>
</html>
